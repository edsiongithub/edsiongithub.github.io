hexo 安装及主题使用

阅读数：次 2021-05-12
字数统计: 554字   |   阅读时长≈ 2分

安装hexo

前提需要机器中安装了node.js（下载最新版）和git（下载最新版），安装过程参考网上教程（widowns下一路无脑下一步）。其中nodejs作为本地运行调试博客程序所必须，当需要将博客部署至github上托管时，需要用到git工具。

检查node是否安装成功

接下来安装hexo，powershell命令行上输入 npm install hexo

安装完成后，创建一个目录，来存放你的博客内容。然后在这个目录下初始化

写博客并进行本地预览

输入如下命令，启动本地调试，当看到一条INFO输出时，说明你的本地环境已经可以了。在浏览器上输入localhost:4000就可以预览你的博客内容了。

修改样式

首先在github上找到喜欢的开源样式（向作者表示感谢）,一般作者都会告诉你如何使用。剩下的就是克隆项目至自己的博客下themes目录中，接着修改_config.yml等配置文件。
如我这里用的yilia样式，github地址 https://github.com/litten/BlogBackup/tree/master/themes/yilia 。克隆至自己的博客目录中，修改_config.yml中theme值为yilia即可。其他配置参考作者给的意见。

使用hexo部署至github

下载安装工具，修改_config.yml底部关于deploy相关的参数，把github仓库地址给写上

添加google搜索

  1. 下载一个google提供的html文件，放到博客根目录上，提交至服务器。Google Search Console上正常访问到这个文件即可。注意每次提交时，别把这个文件给删掉了。
